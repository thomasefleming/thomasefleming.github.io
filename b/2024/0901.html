Data Analysis Resources

September 1, 2024 | Thomas E. Fleming

The Economist shares some of its latest data driven journalism, and sometimes even some code, in their weekly Off the Charts newsletter.

In the August 27, 2024 edition, Marie Segger includes tips using the application R as well as a link to Norm Matloff's "no-thrills" R tutorial. Marie also shared a link to The Economist's Github page which hosts the source code for their data visualisations of the war in Ukraine and other topics. These links are all in the Resources section below.

Also below are some of my Bluesky lists of data accounts. Check my Bluesky profile for more data lists and lists covering other topics. In the list description, I also include a cross-link to the corresponding list on X; Bluesky is growing fast but unlike Mary Segger and much of The Economist's data team, not everyone has moved over to the platform yet.
